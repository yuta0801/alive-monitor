# Discord bot alive monitor

Monitor bots and notify you when offline

## Install

1. `git clone https://github.com/yuta0801/alive-monitor/`
2. `cd alive-monitor`
3. `cp .env.example .env`
4. Edit .env
5. Install dependencies. `yarn`
6. Run. `node index.js`
